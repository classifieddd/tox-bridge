Tox Bridge

Un bot Telegram pour parler sur le réseau Tox.
Pas de compte, pas de serveur, pas de logs.

@pursiveBot

▶ Accès via le réseau Tor

Le problème

Tes messages passent par des serveurs. Même "chiffrés", les applis savent à qui tu parles, pendant combien de temps, depuis où. Les gouvernements peuvent demander ces infos aux entreprises. C'est comme si la poste ouvrait chaque enveloppe pour noter l'expéditeur et le destinataire avant de la refermer.

L'idée

Tox est un réseau pair-à-pair (P2P). Il n'y a pas de serveur central. Quand tu envoies un message, il va directement de ton appareil à celui de ton interlocuteur, chiffré de bout en bout. Personne d'autre ne peut le lire, pas même le logiciel.

Le problème, c'est que Tox n'est pas pratique au quotidien. Il faut installer un client lourd, gérer les mises à jour, être devant son PC.

Tox Bridge fait le lien : tu utilises Telegram (que tu as déjà), le bot s'occupe de parler le protocole Tox pour toi. Tu dialogues sur le réseau P2P sans changer d'appli.

Comment ça marche

  1. Envoie /start au bot — il crée ton identité Tox
  2. /myid pour obtenir ton adresse (une clé publique)
  3. /add suivi de l'adresse d'un ami pour l'ajouter
  4. Quand tu écris un message, le bot le chiffre et l'envoie sur le réseau Tox. Ton ami le reçoit directement, sans intermédiaire.

Ce qui est sauvegardé

  • Ton identité Tox (ta clé privée pour rester joignable)
  • La liste de tes amis
  • Le contact sélectionné
  • Rien d'autre. Pas de messages, pas de logs, pas de tracking.

Pas de pub. Pas de base de données marketing. Pas de traceurs.
Juste un bout de code pour causer tranquille.
